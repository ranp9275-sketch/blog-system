## 如何使用这些文章

### 方案 1：直接发布到博客系统

**步骤**
1. 登录博客管理系统
2. 进入仪表盘
3. 点击"新建文章"
4. 复制 Markdown 内容到编辑器
5. 上传配图
6. 设置分类和标签
7. 发布文章

### 方案 2：转换为其他格式

**转换为 PDF**
```bash
manus-md-to-pdf /home/ubuntu/articles/01_ops_aiops.md output.pdf
```

**转换为 HTML**
```bash
# 使用 pandoc 或其他工具
pandoc -f markdown -t html input.md -o output.html
```

### 方案 3：发布到其他平台

**Medium、掘金、知乎等平台**
1. 复制 Markdown 内容
2. 粘贴到对应平台
3. 上传配图
4. 发布
